How to mine ELF?

Looking forward to mining ELF on the mainnet? AELF, the decentralized cloud computing blockchain network, will allow anyone to mine ELF on personal computers and mining rigs. Until mining is enabled, ELF can be earned for free via the Candy System. This program runs for three years. Have fun collecting your ELF! If you want to learn more about scalable nodes, parallel processing, sidechains, and interoparability, feel free to check out the ELF blog.

How to run an AELF Node?

Run your own personal AELF node. To prepare, sign up for advanced Cloud Hosting and enjoy solid uptime. As soon as the AELF mainnet is launched and the documentary is up, MiningElf will publish an easy guide on how to easily set up and maintain your own node.
